# Netflix User Engagement & Churn Analysis

Project Overview
This project analyzes Netflix user engagement and customer churn behavior using Python and Power BI. The goal is to identify patterns affecting customer retention and understand whether churn is influenced by engagement behavior, subscription type, device usage, or content preferences.

The project combines exploratory data analysis (EDA), behavioral segmentation, and interactive dashboard visualization to generate business insights and retention recommendations.

---

-> Business Problem
Netflix faces customer churn, which impacts long-term retention and subscription growth. The objective of this analysis is to understand the factors associated with churn and identify opportunities to improve user engagement and customer retention.

---

-> Objectives
- Analyze Netflix user behavior data
- Identify patterns related to customer churn
- Compare engagement across subscription types and countries
- Visualize KPIs using Power BI
- Generate actionable business recommendations

---
-> Dataset Information
The dataset contains user-level behavioral information including:

- User demographics
- Subscription type
- Device usage
- Watch behavior
- Engagement metrics
- Churn status

---
-> Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Power BI

---

-> Key KPIs
- Total Users
- Churn Rate
- Retention Rate
- Average Watch Time
- Engagement Level
- Watch Sessions Per Week

---

-> Dashboard Features
The interactive Power BI dashboard includes:

- KPI cards
- Engagement vs Churn analysis
- Subscription type analysis
- Country-wise churn analysis
- Interactive slicers and filters
- Dark Netflix-inspired dashboard theme

---

-> Key Insights
- Low-engagement users showed slightly higher churn behavior
- Subscription behavior impacts retention trends
- Watch time varied across subscription types
- User engagement metrics influenced retention more than basic demographics
- Churn patterns were distributed across multiple countries

---

-> Business Recommendations
- Improve personalized recommendations for low-engagement users
- Launch targeted retention campaigns
- Optimize subscription pricing strategies
- Increase user engagement through recommendation algorithms
- Focus on improving platform stickiness and watch behavior
